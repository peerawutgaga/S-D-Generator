Stub and Driver Generator Online Tool

Stub and Driver Generator Online Tool is a tool for creating an stub or driver for Object-Oriented program testing for sequence diagram and class diagram

Instruction

  1. Upload diagram XML file
  2. Select class under test
  3. Choose source code type (Stub or Driver)
  4. Choose language
  5. Edit the source code
  6. Export
